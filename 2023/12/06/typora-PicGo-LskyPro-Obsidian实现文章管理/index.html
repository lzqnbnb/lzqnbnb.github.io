typora+PicGo+LskyPro+Obsidian实现文章管理

typora+PicGo+LskyPro(github)+Obsidian

介绍
  1. typora 是一个 Markdown 格式的编辑器
  2. PicGo 是一个自动上传图片到图床的工具
  3. LskyPro 是一个图床
  4. Obsidian 是一个.md 文档整理关联工具

搭建

图片上传的整体思路
  1. typora 向本地的 36677 端口发送图片上传的请求
  2. PicGo 默认监听本地 36677 端口，并将监听的数据以接口的形式发送到，指定 Lskypro 所在的图床，返回一个链接
搭建流程
  1. 下载并安装node.js
  2. 下载并安装PicGo
  3. typora-偏好设置

利用 docker 搭建 skyPro

数据库方面可以选 sqite

在 PicGo 中安装 lankong 1.1.3

利用 postman 获取 tocken

获得 tocken “x|xxxx……xx”

​ typora 进行测试

改成这样即可成功，图片自动上传到图床

如果不想用自建的图床可以换成 github 图床
