python安装

进入python官网下载python（需要和电脑版本一致的版本，右键计算机-属性查看）

安装python

  1. 打开安装包，勾选add python 3.8 to PATH,点击Customize installation进入下一页
  2. 全选，点击next
  3. 勾选add python to environmnet variables,选择安装路径（默认c盘）,点击install

测试python是否能调用

  1. 在python文件夹下，输入cmd
  2. 进入命令行，输入python
  3. 出现以下界面则安装成功
