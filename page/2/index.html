1.SELECT子句用来选择来自一个或多个源的数据。函数GROUP_CONCAT用于合并同一组里面的列值为一个单一的字符串。

  1. SELECT 1：直接选择数字 1 作为第一列的值，这通常用于测试或只需输出某些固定值的情况。
  2. GROUP_CONCAT(column_name)：从数据库的 INFORMATION_SCHEMA.COLUMNS 表中选择所有以列 table_name 命名为 ‘users‘ 的行，然后使用 GROUP_CONCAT 函数将 column_name 列的所有值合网成一个逗号分隔的长字符串作为第二列的结果输出。
  3. 3 看起来是另一个直接选择的数字，作为第三列的一个固定值。

2.**SELECT 1, GROUP_CONCAT(table_name), 3**：

  • SELECT 1：会在结果集的第一列返回数字 1。这里选择1通常用于测试或固定值输出，不涉及任何数据库数据检索。
  • GROUP_CONCAT(table_name)：GROUP_CONCAT 是一个聚合函数，它将由查询返回的所有 table_name 行串联成一个字符串，每个 table_name 之间默认使用英文逗号（,）作为分隔符。
  • 3：和 SELECT 1 类似，会在列出结果集的第三列返回数字 3。

**FROM information_schema.tables**：

  • 数据库的 information_schema 是一个特殊的 “虚拟” 数据库，提供了数据库的元数据，比如数据库的表、列、索引等，无论数据库中的具体内容是什么。数据库的 tables 视图包括了库中的所有表的信息。

**WHERE table_schema=database()**：

  • table_schema：是 information_schema.tables 视图的一列，它包含了表所在的数据库名称。
  • database()：是一个函数，返回当前的数据库名称，此处作用是捕获当前（调用查询的客户端选择的）数据库的名字，从而 WHERE 条件筛选出该数据库中的所有表。

返回每个匹配的数据库表名，串联成一个逗号分隔的字符串。这个字符串会出现在结果集的第二列，而第一列和第三列分别是硬编码的数字 1 和 3。

3.该 SQL 语句使用 GROUP_CONCAT 函数来查询 users 表，并将每条记录的 id，username 与 password 字段连接在一起，连接字符为 "~"

  • GROUP_CONCAT 是一个聚合函数，其功能是将多个行值连结成一个单独的字符串。
  • id, username, 和 password 是 users 表中的列名。
  • "~" 是连接各个列值时的分隔符。
  1. 对表 users 中的每一行，该查询会取得 id, username 和 password 字段的值。
  2. 然后，它会使用 "~" 作为分隔符，将这些值串联成一个字符串，比如如果某一行的 id 是 1，用户名(username)为 JohnDoe 而密码（password）为 12345，那么这行的输出将会是 "1~JohnDoe~12345"。
  3. GROUP_CONCAT 函数接着将每一行根据上述方式生成的字符连接起来成一个大字符串，大字符串的每部分之间用逗号 , 分隔（这是 GROUP_CONCAT 默认的分隔符）。

结果将返回所有 users 表行的集合，以一个大字符串表现，其中包含用 "~" 分隔并用逗号连接起来的每个用户的 id，username和 password数据。

php学习

php学习

php

PHP 是一种创建动态交互性站点的强有力的服务器端脚本语言

PHP 脚本以 结束：

PHP 文件的默认文件扩展名是 .php

PHP 中的每个代码行都必须以分号结束。分号是一种分隔符，用于把指令集区分开来。

通过 PHP，有两种在浏览器输出文本的基础指令：echo 和 print。

PHP 变量

与代数类似，可以给 PHP 变量赋予某个值（x=5）或者表达式（z=x+y）。

变量可以是很短的名称（如 x 和 y）或者更具描述性的名称（如 age、carname、totalvolume）。

PHP 变量规则：

  • 变量以 $ 符号开始，后面跟着变量的名称
  • 变量名必须以字母或者下划线字符开始
  • 变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ）
  • 变量名不能包含空格
  • 变量名是区分大小写的（$y 和 $Y 是两个不同的变量）

echo 和 print

  • echo - 可以输出一个或多个字符串
  • print - 只允许输出一个字符串，返回值总为 1
  • echo 输出的速度比 print 快， echo 没有返回值，print有返回值1
  • echo print是语言结构，使用的时候可以不用加括号，也可以加上括号： echo 或 echo()

类型比较

PHP 是弱类型语言，对 PHP 变量进行比较，包含松散和严格比较

  • 松散比较：使用两个等号 == 比较，只比较值，不比较类型。
  • 严格比较：用三个等号 === 比较，除了比较值，也比较类型。

PHP 常量

常量是一个简单值的标识符。该值在脚本中不能改变。

一个常量由英文字母、下划线、和数字组成,但数字不能作为首字母出现。

常量名不需要加 $ 修饰符。

常量在整个脚本中都可以使用。

PHP 条件语句

  • if 语句 - 在条件成立时执行代码
  • if…else 语句 - 在条件成立时执行一块代码，条件不成立时执行另一块代码
  • if…elseif….else 语句 - 在若干条件之一成立时执行一个代码块
  • switch 语句 - 在若干条件之一成立时执行一个代码块

PHP 循环

  • while - 只要指定的条件成立，则循环执行代码块
  • do…while - 首先执行一次代码块，然后在指定的条件成立时重复这个循环
  • for - 循环执行代码块指定的次数
  • foreach - 根据数组中每个元素来循环代码块

for 循环

for 循环用于您预先知道脚本需要运行的次数的情况

  • 初始值：主要是初始化一个变量值，用于设置一个计数器（但可以是任何在循环的开始被执行一次的代码）。
  • 条件：循环执行的限制条件。如果为 TRUE，则循环继续。如果为 FALSE，则循环结束。
  • 增量：主要用于递增计数器（但可以是任何在循环的结束被执行的代码）。

foreach 循环

foreach 循环用于遍历数组

每进行一次循环，当前数组元素的值就会被赋值给 $value 变量（数组指针会逐一地移动），在进行下一次循环时，您将看到数组中的下一个值。

每一次循环，当前数组元素的键与值就都会被赋值给 $key 和 $value 变量（数字指针会逐一地移动），在进行下一次循环时，你将看到数组中的下一个键与值。

HTML使用标签 <a>来设置超文本链接

超链接可以是一个字，一个词，或者一组词，也可以是一幅图像，您可以点击这些内容来跳转到新的文档或者当前文档中的某个部分。当你把鼠标指针移动到网页中的某个链接上时，箭头会变为一只小手。

在标签 <a> 中使用了 href 属性来描述链接的地址。在链接之后的文本则是作为链接的显示文本。例如以上则显示为下图所示。

注意：<a> 标签中必须提供 href 属性或 name 属性。

HTML图像：在 HTML 中，图像由<img> 标签定义

<img> 是空标签，意思是说，它只包含属性，并且没有闭合标签。

要在页面上显示图像，你需要使用源属性（src）。src 指 “source”。源属性的值是图像的 URL 地址。

URL 指存储图像的位置。如果名为 “1.jpg” 的图像位于文件夹的 目录中，那么其 URL 为 “./1.jpg”

HTML 部分：

<!DOCTYPE html>：这是文档类型声明，指定文档使用 HTML5 标准。
<html lang="en">：HTML 文档的根元素，lang="en" 属性定义页面语言为英语。
<head>：头部部分包含关于文档的元信息。
<meta charset="UTF-8">：设置文档使用 UTF-8 字符编码，确保能正确显示各种字符。
<meta name="viewport" content="width=device-width, initial-scale=1.0">：配置视口，确保适应移动设备的屏幕。
<title>web2302</title>：定义页面标题为 “web2302”。
<style>：内联样式表，包含了对页面样式的定义。

container 规则：定义页面容器的样式。

width: 500px;：设置容器宽度为 500 像素。
margin:0 auto;：通过设置左右外边距为自动，使容器水平居中。

header 规则：定义页头的样式。

background-color: #FFA500;：设置背景颜色为橙色。
color: #fff;：设置文字颜色为白色。
padding: 10px;：设置内边距为 10 像素。
text-align: center;：文字居中。

menu 规则：定义菜单的样式。

background-color: #FFD700;：设置背景颜色为金色。
height:100px;：设置高度为 100 像素。
width: 500px;：设置宽度为 500 像素

content 规则：定义内容区域的样式。

background-color: #EEEEEE;：设置背景颜色为淡灰色。
height:100px;：设置高度为 100 像素。
width: 500px;：设置宽度为 500 像素。

#ll 和 #ll1 规则：定义内容区域内两个子元素的样式。

background-color: #a74949; 和 background-color: #49a781;：设置不同的背景颜色。
height:50px;：设置高度为 50 像素。
width: 500px;：设置宽度为 500 像素

footer 规则：定义页脚的样式。

background-color: #FFA500;：设置背景颜色为橙色。
color: #fff;：设置文字颜色为白色。
clear: both;：清除两侧浮动元素影响，确保页脚位于容器底部。
text-align: center;：文字居中。
padding: 10px;：设置内边距为 10 像素。
